## jumpserver配置文件参数说明

主要字段如下：  

| 字段                                     | 说明                                 |
|----------------------------------------|------------------------------------|
| SECRET_KEY                             | 敏感字段加密/解密的key                      |
| BOOTSTRAP_TOKEN                        | 组件向Core注册服务时使用的token               |
| CORE_HOST                              | jumpserver项目的url，API请求注册时使用        |
|                                        |                                    |
| SESSION_COOKIE_AGE                     | 用户session cookie的有效期 默认值3600*24    |
| SESSION_COOKIE_DOMAIN                  | 用户session cookie的域                 |
| CSRF_TRUSTED_ORIGINS                   | CSRF的信任源                           |
| CONNECTION_TOKEN_EXPIRATION            | 连接token的有效期                        |
| CONNECTION_TOKEN_EXPIRATION_MAX        | 复用连接token的最大有效期                    |
| SSH_TIMEOUT                            | SSH超时时间                            |
| CLIENT_ALIVE_INTERVAL                  | koko给SSH客户端发送心跳间隔时间，默认30S          |
| RETRY_ALIVE_COUNT_MAX                  | 重复发送心跳探测的最大数                       |
| GUA_HOST                               | GUA主机地址                            |
| GUA_PORT                               | GUA端口                              |
|                                        |                                    |
| SESSION_COOKIE_SECURE                  | session cookie的安全模式，后台只允许https协议传输 |
| CSRF_COOKIE_SECURE                     | csrf cookie的安全模式，后台只允许https协议传输    |
| REFERER_CHECK_ENABLED                  | 是否开启referer校验                      |
| SECURITY_VIEW_AUTH_NEED_MFA            | 需要校验MFA                            |
| CHANGE_AUTH_PLAN_SECURE_MODE_ENABLED   | 改密计划的安全模式，启用后不支持自己该自己              |
